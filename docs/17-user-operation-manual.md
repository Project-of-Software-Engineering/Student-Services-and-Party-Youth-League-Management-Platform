# 用户操作手册

本文档面向课程演示、评审复现和交付验收，说明学生、教师/管理员、领导三类用户的主要操作路径。

## 2. 学生端操作

### 2.1 查看个人工作台

1. 使用学生账号登录。
2. 进入“学生端”。
3. 查看当前用户、当前阶段、画像标签数、提醒数量、未读通知等摘要卡片。
4. 查看党团流程时间线和阶段提醒。

### 2.2 使用政策检索与问答

1. 在“政策检索”输入关键词，例如 `党团`、`奖学金`、`考勤`。
2. 查看系统返回的政策指引。
3. 查看来源文件、分类、版本和摘要依据。
4. 在下方政策卡片查看对应政策要点。

### 2.3 提交线上审批

1. 在“线上申请与审批”区域填写申请类型和申请理由。
2. 点击“创建并提交”。
3. 查看提交后的审批状态和当前节点。
4. 审批被通过、驳回或退回后，在通知和审批列表中查看结果。

### 2.4 提交公开画像变更申请

1. 在“公开画像变更申请”区域填写简介、标签、荣誉、竞赛和实践经历。
2. 点击“提交画像变更”。
3. 在申请记录中查看待审核、已通过或已驳回状态。
4. 管理员或教师审核通过后，正式画像会更新。

### 2.5 查看通知和电子证明

1. 在通知区域查看站内通知。
2. 对未读通知点击“标记已读”。
3. 在“我的电子证明”区域查看已生成证明。
4. 对有效证明点击“下载证明”。

## 3. 教师/管理员端操作

### 3.1 查看管理总览

1. 使用 `demo.admin` 或 `demo.teacher` 登录。
2. 进入“管理端”。
3. 查看用户数、学生数、最近日志、政策数、已发布通知等摘要。
4. 使用管理端二级入口切换学生导入、政策维护、通知发布、电子证明和操作日志。

### 3.2 学生数据导入导出

1. 进入 `/admin/students`。
2. 点击“下载导入模板”，获得 `students-import-template.xlsx`。
3. 按模板填写学生数据后选择 Excel 文件。
4. 点击执行 Excel 导入，查看导入结果和错误提示。
5. 点击“导出学生数据”，下载学生台账。
6. 如需结构化调试，可使用 JSON 导入区域。

### 3.3 审核学生画像变更

1. 进入 `/admin/students`。
2. 查看“画像变更审核”区域。
3. 对待审核申请点击“通过”或“驳回”。
4. 审核通过后，学生正式画像更新。
5. 操作记录写入日志。

### 3.4 维护政策知识库

1. 进入 `/admin/policies`。
2. 填写标题、分类、版本、来源文件名和正文摘要/政策要点。
3. 可选择 PDF、Word、Excel、图片、TXT 或 MD 附件。
4. 点击“新增政策”或“上传政策附件”。
5. 对已有政策可执行编辑、复制新版、启用、停用。
6. 点击“导出政策台账”，下载政策 Excel 台账。

### 3.5 发布通知

1. 进入 `/admin/notices`。
2. 填写通知标题、内容。
3. 按目标标签、年级、专业、班级、政治面貌设置接收范围。
4. 筛选条件留空时面向全部学生。
5. 点击“发布通知”。
6. 在发布记录中查看接收人数、已读人数和未读人数。

### 3.6 管理电子证明

1. 进入 `/admin/certificates`。
2. 在“证明模板管理”中创建模板，填写模板名称、类型编码、模板内容和变量字段。
3. 在“生成电子证明”中选择模板和学生。
4. 点击“生成证明”。
5. 在证明列表中查看编号、学生、状态和签发时间。
6. 对不再有效的证明点击“撤销”。

### 3.7 审计操作日志

1. 进入 `/admin/logs`。
2. 按操作类型、对象类型、开始时间、结束时间筛选。
3. 使用上一页/下一页查看日志。
4. 点击“导出日志”下载 Excel。
5. 重点检查学生导入、政策维护、通知发布、审批、证明生成和导出类操作。

### 3.8 维护班团组织

1. 管理员、教师或班团骨干登录。
2. 进入 `/admin/organizations`。
3. 填写组织名称、年级、专业、班级、负责人、联系方式、组织说明和近期工作计划。
4. 维护团员数、入党积极分子、志愿服务骨干等组织概况。
5. 点击“新增组织”或“保存组织”。
6. 班团骨干账号仅能维护本人负责班级，管理员和教师可维护全部班团组织。

### 3.9 配置业务模板

1. 使用管理员或教师账号登录。
2. 进入 `/admin/templates`。
3. 填写模板名称、分类、业务类型编码、模板文件名、模板说明和内容摘要。
4. 点击“新增模板”或“保存模板”。
5. 在模板台账中可编辑、启用或停用模板。

## 4. 领导端操作

1. 使用 `demo.leader / demo1234` 登录。
2. 进入“领导端”。
3. 查看审批摘要和待处理队列。
4. 查看学生规模、班团组织、业务模板、政策文档、电子证明、通知触达和政治面貌分布。
5. 对流转到领导节点的审批进行处理。
6. 查看审批结果是否同步回学生端和日志。

## 5. 演示建议流程

1. 管理员登录，导入或查看学生数据。
2. 管理员维护政策知识库，并导出政策台账。
3. 学生登录，检索政策并提交审批。
4. 教师或管理员处理前置审批节点。
5. 领导登录，处理领导终审节点。
6. 管理员生成电子证明。
7. 学生下载本人证明。
8. 管理员查看并导出操作日志。

## 6. 当前限制

- 政策问答属于检索式问答，不是大模型语义问答。
- 电子证明当前以 HTML 下载为主，不是正式 PDF 签章文件。
- 管理端复杂操作更适合桌面端，手机端主要建议演示学生端。
- 权限配置页面、字段级脱敏、自动化权限测试仍是后续增强项。

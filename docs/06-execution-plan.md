# 执行计划

## 1. 责任角色

| 角色 | 默认负责人 | 职责 |
| --- | --- | --- |
| 前端负责人 | FE Owner | 页面、路由、状态管理、组件复用、交互体验 |
| 后端负责人 | BE Owner | API、鉴权、业务逻辑、日志、文件处理 |
| 数据负责人 | DBA Owner | 表结构、迁移、初始化数据、备份恢复 |
| 运维负责人 | DevOps Owner | 本地编排、环境变量、部署脚本、运行监控 |
| 测试文档负责人 | QA/Docs Owner | 测试用例、验收清单、接口文档、使用说明 |

## 2. 里程碑

| 里程碑 | 目标 | 完成标志 | 责任角色 |
| --- | --- | --- | --- |
| M0 | 基础成型 | 前后端能跑通，本地环境可启动 | FE Owner / BE Owner / DevOps Owner |
| M1 | P0 闭环 | 知识库、问答、党团流程、日志可演示 | FE Owner / BE Owner / DBA Owner |
| M2 | P1 闭环 | 审批、画像、通知可演示 | FE Owner / BE Owner |
| M3 | 联调验收 | 数据、权限、日志、上传导出联调完成 | 全体角色 |
| M4 | 发布准备 | 文档、测试、部署、备份齐全 | QA/Docs Owner / DevOps Owner |

## 3. 任务清单

### 3.1 基础设施

| 任务 | 负责人 | 交付物 | 依赖 |
| --- | --- | --- | --- |
| 初始化前端工程 | FE Owner | Vue 3 工程骨架 | 无 |
| 初始化后端工程 | BE Owner | NestJS 工程骨架 | 无 |
| 初始化数据库与迁移 | DBA Owner | Prisma schema、初始迁移 | 后端工程 |
| 配置本地编排 | DevOps Owner | Docker Compose、环境变量模板 | 后端工程 |
| 建立文档入口 | QA/Docs Owner | docs 索引与总览 | 无 |

### 3.2 P0 任务

| 任务 | 负责人 | 交付物 | 依赖 |
| --- | --- | --- | --- |
| 政策文档上传 | BE Owner / FE Owner | 上传接口、上传页、文件记录 | 基础设施 |
| 政策检索与问答 | BE Owner / FE Owner | 检索接口、问答页、结果展示 | 政策文档上传 |
| 党团流程展示 | FE Owner / BE Owner | 流程页、阶段状态接口 | 基础数据 |
| 节点提醒 | BE Owner / FE Owner | 提醒规则、提示组件 | 流程数据 |
| 操作日志 | BE Owner / DBA Owner | 日志表、写入与查询接口 | 鉴权骨架 |

### 3.3 P1 任务

| 任务 | 负责人 | 交付物 | 依赖 |
| --- | --- | --- | --- |
| 审批流 | BE Owner / FE Owner | 审批单、流转、退回、重批 | RBAC、日志 |
| 学生画像 | FE Owner / BE Owner | 画像页、基础信息与荣誉展示 | 学生表结构 |
| 通知发布 | BE Owner / FE Owner | 通知录入、标签筛选、发布记录 | 角色与标签 |
| 附件与导出 | BE Owner / DevOps Owner | 导出接口、附件存储 | 文件服务 |

### 3.4 加强项

| 任务 | 负责人 | 交付物 | 依赖 |
| --- | --- | --- | --- |
| 权限细化 | BE Owner | 角色、数据范围、接口守卫 | 登录体系 |
| 统计看板 | FE Owner / BE Owner | 领导端统计页、聚合接口 | 业务数据 |
| 测试用例 | QA/Docs Owner | 核心流程测试清单 | 功能完成 |
| 部署说明 | DevOps Owner / QA/Docs Owner | 部署手册、备份恢复说明 | 环境定稿 |

## 4. 验收标准

- 功能可演示
- 数据可保存
- 权限可区分
- 日志可追溯
- 文档可维护
- 部署可复现

## 5. 执行原则

- 先打通闭环，再做美化
- 先做 P0，再做 P1，P2 只保留占位
- 所有接口先文档化，再实现
- 所有写操作必须有日志

# 前端实施计划

## 1. 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- TanStack Query
- Element Plus

## 2. 页面结构

- 登录页
- 学生端首页
- 政策知识库
- 党团流程
- 审批中心
- 通知中心
- 学生画像
- 管理后台
- 领导看板
- 系统设置

## 3. 前端分层

- `pages`：页面级视图
- `features`：按业务模块拆分
- `components`：通用组件
- `stores`：全局状态
- `services`：接口访问
- `types`：类型定义

## 4. 重点能力

- 表单校验
- 表格分页与筛选
- 文件上传
- 角色切换
- 消息提示
- 审批状态展示

## 5. 前端约束

- 不把业务数据硬编码在组件里
- 不把接口请求散落在页面文件中
- 统一错误提示与加载态
- 统一权限控制和路由守卫

## 6. 交付顺序

1. 登录与基础布局
2. 学生端 P0 页面
3. 管理端审批与通知
4. 领导看板与统计
5. P1/P2 页面补齐

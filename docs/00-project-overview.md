# 项目总览

## 1. 项目定位

学院学生综合服务与党团管理平台，目标是围绕高频事务建立一个网页优先、可维护、可追踪的一站式服务平台。

## 2. 核心范围

- P0：政策知识库与智能问答、党团流程展示与节点提醒、数据管理、权限与日志
- P1：审批流与电子证明、学生画像与荣誉展示、通知发布与精准推送
- P2：学业分析与预警

## 3. 建设原则

- 网页优先，兼容手机与桌面
- 先做闭环，再做扩展
- 数据可维护，规则可配置
- 敏感信息分级授权并留痕

## 4. 目标角色

- 学生
- 班主任 / 辅导员 / 团委老师
- 学院领导
- 系统管理员

## 5. 成功标准

- 学生能完成查询、申请、通知接收、进度查看
- 教师能完成审批、发布、导入、维护、统计
- 系统能记录关键操作并支持后续扩展
